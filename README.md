# 졸꾸러기 지수
2019년 1회 빡독(2019년 02월 16일) 행사에서 스피치한 source code를 공유합니다.

## 목적
뼈아대, 일취월장, 완공 책을 읽고 익힌 것을 일상에서 얼마나 생각하고 있는지 정량적으로 알아보기 위함.  

## 결과
평소에 부족한 topic을 알게 되어 스스로 피드백을 할 수 있었음.
